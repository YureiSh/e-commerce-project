# E-Commerce FullStack Project

>Live Demo: https://e-commerce-project-yureish.vercel.app

Modern, ölçeklenebilir ve güvenli bir e-ticaret uygulaması.
Frontend ve backend ayrı mimaride geliştirilmiş olup, JWT tabanlı authentication ve state management ile gerçek dünya senaryoları hedeflenmiştir.

### Backend

- Java
- Spring Boot
- Spring Security
- JWT (Authentication & Authorization)
- PostgreSQL
- RESTful API

### Frontend

- React
- Redux (State Management)
- TailwindCSS
- Axios

Features
- 🔐 JWT tabanlı kullanıcı authentication
- 👤 Kullanıcı kayıt & giriş sistemi
- 🛍️ Ürün listeleme ve detay sayfası
- 🛒 Sepet (Cart) yönetimi
- ❤️ Favorilere ekleme
- 📦 Sipariş oluşturma
- 🔎 Arama ve filtreleme
- 📱 Responsive tasarım
